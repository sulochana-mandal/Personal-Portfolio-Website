# 🌐 Basic HTML Portfolio Website

## 📌 Project Description

This is a simple multi-page portfolio website created using only HTML and CSS.
The website contains different pages that are connected using a navigation bar.

## 🚀 Features

* 🏠 Home page (About Me & Skills)
* 📂 Projects page (List of projects)
* 📰 Articles page (Basic articles)
* 📞 Contact page (Contact details)
* 🔗 Navigation bar linking all pages
* 🎨 Styled using CSS

---

## 🛠️ Technologies Used

* HTML
* CSS
